# MonsterHunterWorld.CT  
这是一个用于怪物猎人世界的CheatEngine的.ct文件，本人在取得原作者授权后将其翻译至中文。  
汉化是个过程，计划翻译.ct大部分表面内容的名称，**目前尚未完全翻译完成**。  
如果翻译后的文本使您感到混淆，可以试着回退到原始版本。若发现翻译错误或原作者有更新想提醒我一起更新，或者其他任何想法，请在Issue中提交。  
应原作者要求，上传的翻译文件需随附着原始文件。  
如果您喜欢，请支持原作者。
